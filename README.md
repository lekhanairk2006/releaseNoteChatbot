# releaseNoteChatbot
A beginner-friendly RAG chatbot that lets you upload documents and ask questions using local AI — built with FastAPI, Streamlit, ChromaDB, and Ollama.

ReleaseNote Chatbot is a Retrieval-Augmented Generation (RAG) 
system built for beginners to understand how modern AI applications 
work end to end. Upload any PDF, TXT, CSV, or Excel document and ask 
natural language questions about it. The app intelligently retrieves 
the most relevant parts of your document and generates accurate, 
grounded answers using a fully local, free AI stack — no API keys required.
