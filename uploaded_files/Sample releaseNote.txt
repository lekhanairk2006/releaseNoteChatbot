RELEASE NOTES - Version 2.0.0
Product: DocuChat Enterprise
Release Date: June 2026
Status: General Availability

OVERVIEW
Version 2.0.0 introduces major improvements to document processing speed,
user interface, and AI response accuracy. This release focuses on enterprise
readiness and scalability.

NEW FEATURES
1. Multi-document Support
   Users can now upload and query multiple documents simultaneously.
   Switch between documents without re-uploading.

2. Dark Mode
   Full dark mode support across all pages and components.
   Automatically follows system preference.

3. Faster Search
   Vector search is now 3x faster due to index optimization.
   Average query response time reduced from 4s to 1.2s.

4. Export Answers
   Users can now export Q&A sessions as PDF or CSV.

BUG FIXES
- Fixed crash when uploading files larger than 10MB
- Fixed incorrect page numbers in PDF extraction
- Fixed session timeout not clearing uploaded files
- Fixed CSV files with special characters causing encoding errors

KNOWN LIMITATIONS
- Excel files with merged cells may not parse correctly
- Scanned PDFs without OCR are not supported yet
- Maximum file size is 25MB

BREAKING CHANGES
- API endpoint /api/v1/upload is now /api/upload
- Authentication token format has changed, re-login required

UPGRADE INSTRUCTIONS
1. Back up your existing data
2. Run: pip install -r requirements.txt
3. Run database migration: python migrate.py
4. Restart all services
